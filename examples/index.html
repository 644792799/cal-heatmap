Blank calendar

Display 12 hours, from the current hour.

CalHeatMap.init({
		loadOnInit : false
	});

Calendar with custom date domain

Display 6 hours, from midnight on January 1st, 2000.

CalHeatMap.init({
		loadOnInit : false,
		start : new Date(2000, 0, 1),
		id : "graph_a",
		range : 6
	});

Calendar with other range than minutes

Display 15 days, subdivized by hour, from today

CalHeatMap.init({
		loadOnInit : false,
		id : "graph_a",
		range : 6
	});
